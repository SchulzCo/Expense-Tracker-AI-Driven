Expense Tracker is a full-featured application designed to streamline personal finance management. Leveraging Artificial Intelligence for receipt processing, it provides an interactive dashboard and robust tools for tracking, analyzing, and optimizing expenses.
🚀 Key Features
- AI-Powered OCR: Automated extraction of amount, description, date, and category from receipts using OpenAI.
- Interactive Dashboard: Expense visualization with dynamic charts (Recharts), monthly comparisons, and budget alerts.
- Expense Management: Complete CRUD functionality for categorized transactions.
- Data Export: Generate and download expense reports in CSV format.
- Secure Authentication: Integrated with Firebase Authentication.
- Cross-Platform Architecture: Backend powered by FastAPI and a modern Next.js frontend.
🛠️ Technology Stack
Backend
- FastAPI (Python 3.10+)
- PostgreSQL with SQLAlchemy
- Firebase Admin SDK for authentication
- OpenAI API for OCR
- SQLAlchemy + Psycopg ORM
Frontend
- Next.js 15+ (App Router)
- TypeScript
- Tailwind CSS 4
- Recharts for data visualization
- React Hooks + Firebase Client SDK for state management
📋 Prerequisites
- Python 3.10+
- Node.js 20+ with pnpm (or npm)
- PostgreSQL running locally
- Firebase account for authentication
- OpenAI API key for OCR
